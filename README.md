# TimeDesk

A clean, minimal web app that provides a **Clock**, **Timer**, and **Stopwatch**.  
Includes a launcher page with a left side panel that opens each tool in a separate browser tab, plus individual pages for each tool. Ready for GitHub Pages.

## Demo
Live demo: https://6a6258c351cca1cc83bbe373--funny-buttercream-93522c.netlify.app/

## Features
- Launcher with minimize / maximize behavior
- Clock page with large bold 24‑hour display, centered
- Timer page accepts minutes, large centered display, Start and Reset
- Stopwatch page with Start Stop Reset and large centered display
- Each tool opens in its own browser tab from the launcher
- Dark theme and easy font customization
